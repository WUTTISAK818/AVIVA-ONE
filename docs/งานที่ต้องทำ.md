# งานที่ต้องทำ (Backlog)

รวมงานที่รอพัฒนาต่อ — อัปเดตเมื่อมีงานใหม่หรือทำเสร็จ

---

## ต่อยอดการพัฒนา LINE

> สถานะปัจจุบัน (ถึง v4.79): ผูกบัญชี LINE, แจ้งเตือนเข้า LINE ส่วนตัว,
> และ AI ดักจับบทสนทนาลูกค้า → สรุป/เด้งงานให้เซล ทำงานแล้ว

งานที่รอทำต่อ:

- [ ] **หน้าดูบทสนทนา LINE ในแอป** — รวมแชตลูกค้าจาก `line_conversations`
      ดูรายคน + ประวัติ + AI สรุป (ตอนนี้ข้อมูลถูกเก็บแล้วแต่ยังไม่มีหน้าแสดง)
- [ ] **รายงานวิเคราะห์ภาพรวม** — AI สรุปรายวัน/สัปดาห์: ลูกค้าถามอะไรบ่อย,
      จุดที่เสียลูกค้า, ข้อเสนอปรับปรุง OA / สคริปต์ขาย
- [ ] **จับคู่ลูกค้า LINE → lead** — ปุ่มในแอปเชื่อม `line_user_id` เข้า `leads`
      เพื่อ route งานให้เซลเจ้าของแบบเจาะจง (แทนการเด้งหาทุกคน)
      + ลองจับคู่อัตโนมัติจากเบอร์โทร/ชื่อ
- [ ] **กลุ่มแจ้งเตือนทีมขาย** — ตั้ง `line_sale_group_id` (พิมพ์ `/id` ในกลุ่มเพื่อเอา Group ID)
      ให้เด้งเข้ากลุ่มเดียวแทนส่งรายคน (ประหยัดโควต้า push ของ LINE)
- [ ] **บอตตอบลูกค้าอัตโนมัติ (ถ้าต้องการ)** — AI ตอบ FAQ พื้นฐานนอกเวลาทำการ
      (เสี่ยงกว่า — ต้องคุมสโคป/มี human handoff)
- [ ] **กันเด้งถี่ (debounce)** — รวมข้อความลูกค้าที่ส่งรัวๆ เป็นก้อนเดียว
      ก่อนเรียก AI + เด้งเซล (ลดสแปม + ประหยัดโทเคน)
- [ ] **ติดตามโควต้า LINE push** — เฝ้าระวังจำนวนข้อความ push ต่อเดือน (free tier มีลิมิต)

### บันทึกข้อจำกัดที่รู้แล้ว
- LINE Messaging API **อ่านแชตย้อนหลังไม่ได้** — ดักจับได้เฉพาะข้อความใหม่ผ่าน webhook
- ข้อความที่เซล/แอดมินตอบลูกค้าผ่าน **OA Manager โดยตรง ไม่เข้า webhook** — เก็บได้เฉพาะขา "ลูกค้าส่งเข้า"
