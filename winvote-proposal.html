ปรับขนาดอักษร: กกกก

1 UI สำหรับผู้สูงอายุ

  • โหมดสว่างเป็นค่าเริ่มต้น + ตัวเลือกคอนทราสต์สูง (ตัวอักษรเข้ม พื้นสว่าง อ่านง่ายกลางแดด) ปรับ
  • ปุ่มปรับขนาดตัวอักษร 4 ระดับ (เล็ก/กลาง/ใหญ่/ใหญ่พิเศษ) จำค่าไว้ในเครื่อง เพิ่ม
  • ปุ่มใหญ่ เป้าแตะ ≥ 48px, ระยะห่างเยอะ, ไอคอนใหญ่+ข้อความกำกับ
  • สีสื่อสถานะชัด: เขียว=ครบ ทอง=ใกล้ แดง=ต่ำ (เหมือนที่ใช้อยู่)
  • ลดขั้นตอน/หน้าจอซับซ้อน, ปุ่มหลักใหญ่ชัดบนสุด

2 ใช้งานตอนเน็ตหลุด (ออฟไลน์)

  • บันทึกข้อมูลได้แม้ไม่มีเน็ต — เก็บลงเครื่อง (IndexedDB) เข้าคิวรอ เพิ่ม
  • Sync อัตโนมัติเมื่อเน็ตกลับมา + แถบสถานะ "ออฟไลน์ · รอส่ง X รายการ"
  • PWA ติดตั้งลงหน้าจอโทรศัพท์ เปิดเร็ว ใช้เหมือนแอป
  • โหลดข้อมูลพื้นที่ที่รับผิดชอบไว้ล่วงหน้า (หน่วย/ชุมชน) เปิดดูออฟไลน์ได้
  • กันข้อมูลหายถ้าปิดแอป/แบตหมด (เซฟร่างอัตโนมัติ)

3 ล็อกอินผูกอุปกรณ์ (กันส่งต่อรหัส)

  • 1 บัญชี = 1 เครื่อง — ผูกรหัสอุปกรณ์ครั้งแรกที่ล็อกอิน เพิ่ม
  • ถ้านำรหัสไปล็อกอินเครื่องอื่น → ถูกบล็อก ต้องให้ผู้ดูแลอนุมัติ/รีเซ็ตเครื่องก่อน
  • ยืนยันตอนผูกเครื่องใหม่ด้วย OTP ทาง LINE/SMS
  • บังคับ session เดียว (ล็อกอินที่ใหม่ = เตะเครื่องเก่าออก)
  • เก็บ log การเข้าใช้ (ใคร/เมื่อไหร่/เครื่องไหน) ตรวจย้อนหลังได้
🔒 ผลลัพธ์: ส่งลิงก์/รหัสให้คนนอกก็เข้าไม่ได้ — เข้าได้เฉพาะเครื่องที่อนุมัติแล้วเท่านั้น ป้องกันข้อมูลลับรั่วไปคู่แข่ง

4 สิทธิ์การเข้าถึง 5 ชั้น

บทบาทเห็นทุกเขตแก้ข้อมูลอนุมัติ/ยืนยันจัดการผู้ใช้ส่งออก/รายงาน
ผู้บริหารระดับสูง✓✓✓✓✓
เจ้าหน้าที่ระดับสูง✓✓✓✗✓
เจ้าหน้าที่ระดับกลางเฉพาะเขตตน✓บางส่วน✗เขตตน
ผู้ใช้ระดับสูงชุมชนตน✓✗✗✗
ผู้ใช้ระดับกลางของตนเพิ่มเท่านั้น✗✗✗
✓ = ได้ · ✗ = ไม่ได้ · ระบุขอบเขต = จำกัดตามเขต/ชุมชนที่รับผิดชอบ
บังคับจริงด้วย Row-Level Security (RLS) ที่ฐานข้อมูล — แต่ละคนเห็น/แก้ได้เฉพาะข้อมูลในขอบเขตตน แม้จะพยายามดึงตรงก็ไม่หลุด ปรับจากระบบ role เดิม

5 ยืนยัน "ผู้สนับสนุนตัวจริง" (กันปลอม/หลอกใช้)

สแกนบัตร ปชช.→ เช็คเลข 13 หลัก + กันลงซ้ำ→ เซลฟี่ + GPS + เวลา→ ยืนยันเบอร์ (OTP/LINE)→ ✓ ตัวจริง (verified)
  • กันลงซ้ำ: เลขบัตร 13 หลักไม่ซ้ำ + เตือนชื่อ/เบอร์ซ้ำ มีพื้นฐานแล้ว
  • พิสูจน์ลงพื้นที่จริง: เซลฟี่ + พิกัด GPS + เวลา → กันกรอกมั่ว/นั่งโต๊ะปั๊มข้อมูล
  • ยืนยันเบอร์เป็นเจ้าของจริง: OTP ทาง LINE/SMS
  • สถานะความน่าเชื่อถือต่อคน: ยังไม่ยืนยัน / ยืนยันบางส่วน / ยืนยันเต็ม
  • นับเข้าเป้า 10,000 เฉพาะ "ยืนยันเต็ม" → ตัวเลขที่เห็นคือของจริง ไม่ใช่ตัวเลขลม
  • ตั้งเกณฑ์ได้: กำหนดว่าต้องครบกี่อย่างถึงนับเป็นตัวจริง (เช่น บัตร+เซลฟี่+เบอร์)
  • ตรวจสอบย้อนหลัง: ใครเก็บ/ที่ไหน/เมื่อไหร่ — ตามรอยได้ทุกราย
